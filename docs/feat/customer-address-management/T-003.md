# T-003 — Create Address DTOs

**Feature:** Customer Address Management
**Layer:** Application Service
**Linked Story:** US-001
**Estimate:** 1h

## Description

Define the Data Transfer Objects (DTOs) for address operations: CreateAddressDto, UpdateAddressDto, AddressDto, and AddressListDto (for pagination). Include proper validation attributes and data annotations.

---

## Acceptance

- CreateAddressDto with all required fields (no Id), CustomerId, and validation attributes
- UpdateAddressDto with Id for update, optional fields (partial update support)
- AddressDto for responses with all address fields including Id and audit fields
- AddressListDto for paginated results with TotalCount and Items collection
- DTOs include appropriate data annotations for validation (Required, StringLength, etc.)
- DTOs are properly documented with XML comments
