## Description

Create the RESTful API controller with endpoints for address CRUD operations. Follow ABP's controller conventions, use proper HTTP verbs, status codes, and route naming.

---

## Acceptance

- AddressController created with [ApiController] and [Route("api/app/addresses")]
- POST /api/app/addresses — create address (maps to CreateAsync)
- PUT /api/app/addresses/{id} — update address (maps to UpdateAsync)
- DELETE /api/app/addresses/{id} — delete address (maps to DeleteAsync)
- GET /api/app/addresses/{id} — get single address (maps to GetAsync)
- GET /api/app/addresses — get list with pagination (Skip, MaxResultCount) (maps to GetListAsync)
- Controller uses IAddressAppService injected via constructor
- Responses follow ABP's standard format with result, success, etc.
- Proper HTTP status codes returned (201 Created, 200 OK, 204 No Content, 404 Not Found)
- [FromServices], [FromBody] attributes applied correctly
