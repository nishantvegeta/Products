# T-017 — Grant Order Processor Read Access to Customer Addresses

**Feature:** Customer Address Management
**Layer:** Permissions
**Linked Story:** US-005
**Estimate:** 0.5h
**Status:** Open

---

## Description

Define and assign permission for order processors to view customer addresses as needed for order fulfillment. This may be part of existing order management permissions or a new dedicated permission. Ensure the permission system allows read access without allowing modification.

---

## Acceptance

- Permission constant defined: AddressManagement.Address.View (or reuse existing OrderManagement.Read* permissions).
- OrderProcessor role is assigned the view permission for addresses.
- Permission checks are enforced in address-related endpoints accessible to order processors (likely via admin or order controllers).
- Unit tests verify that order processor role can successfully retrieve address data and cannot perform write operations.
- Documentation updated to reflect the new permission assignment.
