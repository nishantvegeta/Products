# T-023 — Enhance Order Management View for Order Processors

**Feature:** Customer Address Management
**Layer:** Frontend — Page / View
**Linked Story:** US-005
**Estimate:** 1.5h
**Status:** Open

---

## Description

Update the order processing/management interface to prominently display billing and shipping addresses on the order detail view. Ensure addresses are clearly shown whether they are saved or one-time. Optionally provide quick access to the customer's full address book for comparison.

---

## Acceptance

- Order detail page includes dedicated sections for Billing Address and Shipping Address.
- Distinguish between "Saved Address" (with label/icon) and "One-Time Address" (manually entered).
- Display complete address fields in a readable format.
- If saved address was used, provide a link or button to view the customer's address book (read-only).
- Print/export functionality for address information is available (or notes that standard print covers it).
- Page respects order processor permissions (only accessible to authorized roles).
- Layout is clean and fits within existing order management UI.
