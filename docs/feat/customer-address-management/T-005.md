## Description

Define the IAddressRepository interface with methods for CRUD operations and queries specific to address management. Include methods for retrieving addresses by customer, finding default addresses, and checking exclusive default constraints. Follow the repository pattern used in the project.

---

## Acceptance

- IAddressRepository interface extends IRepository<Address, Guid> (or TKey).
- Custom methods added: GetByCustomerIdAsync(customerId), GetDefaultBillingAsync(customerId), GetDefaultShippingAsync(customerId), SetDefaultBillingAsync(addressId), SetDefaultShippingAsync(addressId).
- All methods are asynchronous and return appropriate results.
- Interface is placed in Domain layer with other repository contracts.
- Unit tests verify interface contract is correctly defined.
