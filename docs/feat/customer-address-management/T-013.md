# T-013 — Implement Admin Methods in AddressAppService

**Feature:** Customer Address Management
**Layer:** Application Service
**Linked Story:** US-002
**Estimate:** 1.5h

---

## Description

Implement the admin methods in AddressAppService with proper permission checks and audit logging. These methods allow administrators to query any customer's addresses and perform managed deletions with reason tracking.

---

## Acceptance

- GetAddressesByCustomerAsync queries repository for all addresses of specified customer, with pagination
- SearchAddressesAsync supports filtering by customer (ID/name), address type, default status
- Admin delete method requires reason parameter, logs to audit trail, performs soft delete
- All admin methods check for admin permission before execution
- Audit logging includes: admin user ID, action type, address ID, reason, timestamp
- Methods return appropriate DTOs with full details
