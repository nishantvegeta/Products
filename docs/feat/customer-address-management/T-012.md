# T-012 — Extend IAddressAppService with Admin Methods

**Feature:** Customer Address Management
**Layer:** Application Service
**Linked Story:** US-002
**Estimate:** 1h

---

## Description

Add admin-specific methods to IAddressAppService: GetAllAddresses (with customer filter), GetAddressesByCustomerId, and possibly AdminDeleteAsync (with reason). These methods bypass customer ownership restrictions.

---

## Acceptance

- Interface extended with GetAddressesByCustomerAsync(customerId, pagination) method
- Interface possibly extended with SearchAddressesAsync(filters) for admin search
- Methods return address DTOs with full details including owner information
- Admin methods include optional parameters for filtering by address type or default status
- XML documentation added for new methods
