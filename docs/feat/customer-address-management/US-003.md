## Title

[Customer Address Management] Add user story and scenarios for selecting saved addresses when placing an order

---

## User Story

As a customer,
I want to select from my saved billing and shipping addresses when creating a new order,
So that I can checkout faster without re-entering address information.

---

## Scenario 1: Successful flow

Given I have at least one saved address in my address book
When I proceed to the checkout page while creating a new order
Then I see options to either select a saved billing address or enter a new one
And I see options to either select a saved shipping address or enter a new one
When I select a saved address for billing and/or shipping
Then the order form is automatically populated with the selected address details
And I can complete the order without manual address entry

---

## Scenario 2: Validation / error flow

Given I am placing an order and select a saved address
When I proceed to confirm the order
Then the system validates that both billing and shipping addresses are provided
If I have no default addresses set, the system prompts me to select addresses manually
And I cannot complete the order until valid addresses are provided

---

## Scenario 3: Alternate flow

Given I am placing an order
When I choose to enter a new one-time address instead of selecting from my saved addresses
Then I can manually enter billing and/or shipping address details
And I have the option to save that new address to my address book for future use
And the order is created successfully with the entered address information

---

## Acceptance Criteria

- During order creation, the customer can select any saved billing address from their address book.
- During order creation, the customer can select any saved shipping address from their address book.
- The customer can choose to enter a new ad-hoc address instead of using a saved one.
- When a saved address is selected, all address fields are populated automatically into the order form.
- The system respects default address selections but also allows choosing any saved address.
- The customer can optionally save a newly-entered address during checkout.
- The order creation workflow requires both billing and shipping addresses to be provided before submission.
- The system handles the case where the customer has no saved addresses (they must enter addresses manually).
