## Description

Create OrderAppService with CreateOrderAsync method. Logic: validate all products exist and are active, check stock availability, lock prices by copying Product.Price to OrderItem.UnitPrice, calculate TotalAmount, create Order and OrderItems, set status to Draft. Handle transaction and concurrency (RowVersion).

---

## Acceptance

- OrderAppService created
- CreateOrderDto accepted with CustomerId, BillingAddressId, ShippingAddressId, OrderItems list
- Validates product existence, active status, stock quantity
- Locks prices at time of order creation
- Calculates TotalAmount as sum of line totals plus any shipping/tax
- Sets initial status to Draft
- Uses UnitOfWork to ensure atomic creation
- Concurrency control via RowVersion
