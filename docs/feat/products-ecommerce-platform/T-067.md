# T-067 — Implement order export functionality

**Feature:** Products E-commerce Platform
**Layer:** Application Service
**Linked Story:** US-009
**Estimate:** 2h
**Status:** Open

---

## Description

Create ExportService or extend OrderAppService with ExportOrdersAsync method. Accept filters: date range, status. Generate CSV stream with fields: OrderId, OrderNumber, CustomerEmail, OrderDate, Status, TotalAmount. Use efficient streaming for large datasets. Include proper memory management (yield rows).

---

## Acceptance

- ExportOrdersAsync method returns Stream or byte[] containing CSV
- CSV header row included
- Filters applied at database level (not in-memory)
- Streaming implemented to handle large datasets without OOM
- Data includes selected fields only
- Response ready for file download with Content-Disposition
