# T-020 — Implement address default management logic

**Feature:** Products E-commerce Platform
**Layer:** Application Service
**Linked Story:** US-003
**Estimate:** 1.5h
**Status:** Open

---

## Description

In AddressAppService, implement business logic for default address management. When a customer sets IsDefault=true on an address of a given AddressType, automatically unset IsDefault on all other addresses of the same type for that customer. Ensure only one default per type per customer.

---

## Acceptance

- Address creation/update logic enforces single default per AddressType
- Setting IsDefault=true clears the flag on other addresses of same type
- Logic executes within same unit of work (transaction)
- Concurrency handled properly if multiple updates occur
- Customer cannot have two default Billing or two default Shipping addresses
