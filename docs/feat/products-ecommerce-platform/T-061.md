# T-061 — Implement audit logging for user changes

**Feature:** Products E-commerce Platform
**Layer:** Application Service
**Linked Story:** US-008
**Estimate:** 1h
**Status:** Open

---

## Description

Ensure UserManagementAppService uses ABP's built-in audit logging or explicit EntityHistory to record: user creation (who created, when), role changes (which user changed which user's role), and deactivations. Verify audit entries are stored with correct entity type and field changes.

---

## Acceptance

- Audit log configured in module to capture IdentityUser changes
- User creation event logs AdminUserId
- Role change logged with old and new roles
- Soft delete logged
- Audit entries queryable via ABP's audit system
