## Description

Create UsersController with endpoints: GET /api/app/users (list with filters), GET /api/app/users/{id}, POST /api/app/users, PUT /api/app/users/{id}, DELETE /api/app/users/{id} (soft). Apply [Authorize(Roles = "Admin")] to all endpoints. Wire to UserManagementAppService.

---

## Acceptance

- UsersController with full CRUD endpoints
- All require Admin role
- List endpoint supports query filters: SearchEmail, Role, IsActive, IsDeleted, CreatedDateFrom, CreatedDateTo
- Returns UserDto with UserId, Email, Roles, Profile fields, LastLoginTime, IsActive, CreationTime
- Soft delete implemented (IsDeleted flag)
