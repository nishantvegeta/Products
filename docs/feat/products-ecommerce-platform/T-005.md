# T-005 — Implement Product AppService with catalog operations

**Feature:** Products E-commerce Platform
**Layer:** Application Service
**Linked Story:** US-001
**Estimate:** 1.5h
**Status:** Open

---

## Description

Create ProductAppService that inherits from CrudAppService or implements IApplicationService. Expose methods: GetProductListAsync (with pagination, category filter), GetProductDetailAsync, GetProductBySkuAsync. Return appropriate DTOs. Ensure proper authorization is not required for these methods.

---

## Acceptance

- ProductAppService class created in Application layer
- DTOs defined: ProductDto, ProductListDto
- Mapping configuration between Entity and DTO using IObjectMapper
- GetProductListAsync supports pagination, category filtering, active-only filter
- Methods are marked with [AllowAnonymous] attribute
- Response follows ABP wrapper format
