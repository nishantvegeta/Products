# T-016 — Configure password complexity rules

**Feature:** Products E-commerce Platform
**Layer:** Config / Migration
**Linked Story:** US-002
**Estimate:** 0.5h
**Status:** Open

---

## Description

Configure ABP Identity options in module's ConfigureServices to enforce password requirements: minimum length, RequireDigit, RequireUppercase, RequireLowercase, RequireNonAlphanumeric. Ensure these align with acceptance criteria.

---

## Acceptance

- Password options set in IdentityBuilder or ABP configuration
- Requirements: minimum 8 characters, at least one digit, one uppercase, one lowercase, one special character
- Configuration applied at application startup
- Registration validation enforces these rules
