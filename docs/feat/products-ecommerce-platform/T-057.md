## Description

Create UserRepository (or extend IdentityUserRepository) with methods: GetUserListAsync (with filters: email contains, role, isActive, isDeleted, createdDate range), GetUserWithRolesAsync(userId), UpdateUserRolesAsync. Ensure queries are efficient and return required data for admin user list.

## Acceptance

- User repository methods implemented
- GetUserListAsync supports filtering and pagination
- Includes join with Identity user roles or claims to determine role
- GetUserWithRolesAsync returns user with role assignments
- Filter by IsActive, IsDeleted, CreatedDate range
- Email search uses case-insensitive LIKE
