## Description

Design and implement optimized queries for dashboard and reporting. Queries include: total order count, total revenue (sum of completed orders), active customer count, active product count, recent orders (last 10), top selling products (last 30 days by quantity). Use raw SQL or LINQ with proper joins and grouping. Ensure queries use indexes and avoid full scans.

---

## Acceptance

- Repository methods for each dashboard metric implemented
- Single query or combined query returns DashboardSummaryDto efficiently (<2s target)
- Queries join Orders, OrderItems, Products, Customers
- Filters: completed orders for revenue, non-deleted for active counts
- TopSellingProducts grouped by Product and summed quantity in date range
- Performance validated with reasonable data volumes
