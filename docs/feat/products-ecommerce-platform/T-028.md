# T-028 — Define OrderItem entity

**Feature:** Products E-commerce Platform
**Layer:** Data / Entity
**Linked Story:** US-004
**Estimate:** 1.5h
**Status:** Open

---

## Description

Define the OrderItem entity to store line items within an order. Fields: Id, OrderId (FK), ProductId (FK), ProductName (snapshot), UnitPrice (decimal, locked at order time), Quantity, LineTotal (calculated). Ensure FK to Product is maintained per BR-007.

---

## Acceptance

- OrderItem entity created
- Foreign keys: OrderId to Order, ProductId to Product
- ProductName stored as snapshot for historical display
- UnitPrice stored at order time (price lock)
- LineTotal calculated as UnitPrice × Quantity
- Composite unique constraint on (OrderId, ProductId) may be needed
