# T-053 — Add order list endpoint for managers

**Feature:** Products E-commerce Platform
**Layer:** API / Controller
**Linked Story:** US-007
**Estimate:** 1h
**Status:** Open

---

## Description

In OrdersController, add GET /api/app/orders endpoint for manager order list with filters (status, date range, customerId), pagination, and sorting. This is separate from customer's own orders endpoint. Authorization: Manager role only.

---

## Acceptance

- GET /api/app/orders endpoint (manager only)
- Query parameters: Status, MinCreatedDate, MaxCreatedDate, CustomerId, Skip, MaxResultCount, Sorting
- Returns PagedResultDto<OrderListDto> with summarized order info
- OrderListDto includes: OrderId, OrderNumber, CustomerName (or Email), TotalAmount, Status, OrderDate
- Excludes soft-deleted orders
