# T-058 — Implement user management AppService

**Feature:** Products E-commerce Platform
**Layer:** Application Service
**Linked Story:** US-008
**Estimate:** 2h
**Status:** Open

---

## Description

Create UserManagementAppService with methods: GetUserListAsync (with filters, pagination, sorting), GetUserDetailAsync(userId), CreateUserAsync (with email, password, role, profile), UpdateUserAsync (profile, role, active status), SoftDeleteUserAsync. Use ABP's UserManager for Identity operations. Ensure role validation against allowed roles.

---

## Acceptance

- UserManagementAppService created
- CreateUserAsync creates IdentityUser, sets password, assigns role
- UpdateUserAsync updates profile and role (with change audit)
- SoftDeleteUserAsync sets IsDeleted flag (not hard delete)
- All role changes validated to allowed set: Admin, Manager, Customer, Guest
- Self-protection: Admin cannot modify own role to non-Admin or delete self
- Logs all admin actions (audit)
