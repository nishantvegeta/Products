# T-015 — Add registration API endpoint

**Feature:** Products E-commerce Platform
**Layer:** API / Controller

**Estimate:** 1h

## Description

Create AuthenticateController or extend Identity API with POST /api/app/auth/register endpoint. Accept DTO with Email, Password, FirstName, LastName, PhoneNumber. Call RegistrationAppService. Return CustomerDto with JWT token on success (201 Created).

---

## Acceptance

- Controller action created for registration
- Allows anonymous access (no authentication required)
- Input DTO validated with DataAnnotations or FluentValidation
- Returns 201 Created with CustomerDto and authentication token
- Errors return 400 with validation messages
- API follows ABP conventions
