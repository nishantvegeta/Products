# T-014 — Implement registration business logic

**Feature:** Products E-commerce Platform
**Layer:** Application Service
**Linked Story:** US-002
**Estimate:** 2h
**Status:** Open

---

## Description

Create RegistrationAppService (or extend IdentityAppService) to handle user registration. Logic includes: validate email uniqueness, create IdentityUser with password hashing, assign Customer role, create Customer entity, generate JWT token for immediate login, and save changes atomically.

---

## Acceptance

- RegistrationAppService created with RegisterAsync method
- Uses ABP's UserManager for IdentityUser operations
- Email validated for uniqueness before creation
- Password meets complexity requirements
- Customer role assigned automatically
- JWT token generated and returned in response
- Transaction ensures IdentityUser and Customer entity created together
