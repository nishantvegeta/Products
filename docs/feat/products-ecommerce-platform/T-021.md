# T-021 — Create Customer Profile AppService

**Feature:** Products E-commerce Platform
**Layer:** Application Service
**Linked Story:** US-003
**Estimate:** 1h
**Status:** Open

---

## Description

Create CustomerProfileAppService with methods: GetProfileAsync (returns CustomerDto with profile fields), UpdateProfileAsync (accepts UpdateCustomerProfileDto). Restrict access to authenticated user's own data. Exclude email and password from update operations.

---

## Acceptance

- CustomerProfileAppService created
- GetProfileAsync returns CustomerDto with FirstName, LastName, PhoneNumber
- UpdateProfileAsync validates and updates profile fields
- Service ensures CustomerId matches authenticated user
- Email and password cannot be changed through this endpoint
