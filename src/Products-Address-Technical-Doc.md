## 10. Error Handling

**Detected Pattern:** UserFriendlyException with custom messages + try/catch blocks in AppService methods returning `ResponseDataDto<object>`

| Scenario | Handling | HTTP Response |
|---|---|---|
| Validation failure | Throw `UserFriendlyException("message")` before insert/update | 400 Bad Request (ABP exception filter converts to ResponseDataDto with Success=false) |
| Address not found | Return `ResponseDataDto<object>` with Success=false, Code=404 | 404 Not Found |
| Unauthorized access | ABP Authorization middleware | 401 Unauthorized or 403 Forbidden |
| Concurrency conflict | ABP handles concurrency tokens (if implemented) | 409 Conflict |
| Duplicate default for type | Clear existing defaults automatically (business logic) | 200 OK |
| Customer mismatch | Return 403 Forbidden or 404 (security through obscurity) | 403 or 404 |
| Unexpected error | catch Exception, log, throw new UserFriendlyException("An unexpected error...") | 500 Internal Server Error |

**Example Pattern (from CustomerAppService):**

```csharp
public async Task<ResponseDataDto<object>> GetAsync([Required] Guid id)
{
    try
    {
        var address = await _addressRepository.FindAsync(id);

            {
                Success = false,
                Message = "Address not found",
                Data = null,
                Code = 404
            };
        }

        return new ResponseDataDto<object>
        {
            Success = true,
            Message = "Address retrieved successfully",

        };

    catch (UserFriendlyException ex)
    {

        throw new UserFriendlyException("An unexpected error occurred. Please try again later.");
    }
```

## 11. Service Implementation (AppService)

**Location:** `src/Products.Application/Addresses/AddressAppService.cs`

```csharp
using System;
using System.Linq;
using System.Threading.Tasks;
using AutoMapper;
using Microsoft.Extensions.Logging;
using Products.Addresses.Dtos;
using Products.Entities.Addresses;
using Volo.Abp;
using Volo.Abp.Application.Dtos;
using Volo.Abp.Application.Services;
using Volo.Abp.Domain.Repositories;
using Volo.Abp.Linq;

namespace Products.Addresses;

public class AddressAppService : ApplicationService, IAddressAppService
{
    private readonly IRepository<Address, Guid> _addressRepository;
    private readonly ILogger<AddressAppService> _logger;
    private readonly IMapper _mapper;

    public AddressAppService(
        IRepository<Address, Guid> addressRepository,
        ILogger<AddressAppService> logger,
        IMapper mapper)
    {
        _addressRepository = addressRepository;
        _logger = logger;
        _mapper = mapper;
    }

    public async Task<ResponseDataDto<object>> CreateAsync(CreateAddressDto input)
    {
        try
        {
            _logger.LogInformation("CreateAsync called with input: {@Input}", input);
            await ValidateInputAsync(input);

            // Ensure CustomerId is set (should come from logged-in user context)
            // input.CustomerId = _currentUser.GetId(); // via ICurrentUser

            var address = _mapper.Map<CreateAddressDto, Address>(input);
            address.CustomerId = CurrentUser.GetId(); // Get from ABP current user

            // Handle default address logic: if setting as default, clear others
            if (input.IsDefaultBilling || input.IsDefaultShipping || input.IsDefaultDelivery)
            {
                await ClearExistingDefaultsAsync(address.CustomerId,
                    input.IsDefaultBilling,
                    input.IsDefaultShipping,
                    input.IsDefaultDelivery);
            }

            await _addressRepository.InsertAsync(address);

            var dto = _mapper.Map<Address, AddressDto>(address);

            return new ResponseDataDto<object>
            {
                Success = true,
                Message = "Address created successfully",
                Data = dto,
                Code = 200
            };
        }
        catch (UserFriendlyException ex)
        {
            _logger.LogWarning(ex, "Validation error: {Message}", ex.Message);
            throw;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Error creating address: {ErrorMessage}", ex.Message);
            throw new UserFriendlyException("An unexpected error occurred. Please try again later.");
        }
    }

    public async Task<ResponseDataDto<object>> GetAsync([Required] Guid id)
    {
        try
        {
            var address = await _addressRepository.FindAsync(id);
            if (address == null || address.IsDeleted)
            {
                _logger.LogWarning("Address with id {Id} not found.", id);
                return new ResponseDataDto<object>
                {
                    Success = false,
                    Message = "Address not found",
                    Data = null,
                    Code = 404
                };
            }

            // Authorization: ensure customer only accesses their own address
            if (address.CustomerId != CurrentUser.GetId())
            {
                _logger.LogWarning("User {UserId} attempted to access address {AddressId} belonging to {CustomerId}",
                    CurrentUser.GetId(), id, address.CustomerId);
                return new ResponseDataDto<object>
                {
                    Success = false,
                    Message = "Access denied",
                    Data = null,
                    Code = 403
                };
            }

            var dto = _mapper.Map<Address, AddressDto>(address);
            return new ResponseDataDto<object>
            {
                Success = true,
                Message = "Address retrieved successfully",
                Data = dto,
                Code = 200
            };
        }
        catch (UserFriendlyException ex)
        {
            _logger.LogWarning(ex, "Validation error: {Message}", ex.Message);
            throw;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Error retrieving address: {ErrorMessage}", ex.Message);
            throw new UserFriendlyException("An unexpected error occurred. Please try again later.");
        }
    }

    public async Task<ResponseDataDto<object>> GetListAsync(Guid customerId)
    {
        try
        {
            // Ensure customer only requests their own addresses
            if (customerId != CurrentUser.GetId())
            {
                return new ResponseDataDto<object>
                {
                    Success = false,
                    Message = "Access denied",
                    Data = null,
                    Code = 403
                };
            }

            var addresses = await AsyncExecuter.ToListAsync(
                _addressRepository
                    .Where(a => a.CustomerId == customerId && !a.IsDeleted)
                    .OrderByDescending(a => a.IsDefaultBilling)
                    .ThenByDescending(a => a.IsDefaultShipping)

    public async Task<ResponseDataDto<object>> UpdateAsync([Required] Guid id, UpdateAddressDto input)
    {
        try
        {
            var address = await _addressRepository.FindAsync(id);
            if (address == null || address.IsDeleted)
            {
                return new ResponseDataDto<object>
                {
                    Success = false,
                    Message = "Address not found",
                    Data = null,
                    Code = 404
                };
            }

            // Authorization check
            if (address.CustomerId != CurrentUser.GetId())
            {
                return new ResponseDataDto<object>
                {
                    Success = false,
                    Message = "Access denied",
                    Data = null,
                    Code = 403
                };
            }

            // Prevent making all addresses inactive
            if (input.IsActive.HasValue && input.IsActive.Value == false)
            {
                var activeCount = await _addressRepository.CountAsync(
                    a => a.CustomerId == address.CustomerId && a.IsActive && a.Id != id);
                if (activeCount == 0)
                {
                    return new ResponseDataDto<object>
                    {
                        Success = false,
                        Message = "Cannot deactivate the only active address",
                        Data = null,
                        Code = 400
                    };
                }
            }

            // Handle default changes: clear other defaults if setting new ones
            if (input.IsDefaultBilling == true || input.IsDefaultShipping == true || input.IsDefaultDelivery == true)
            {
                await ClearExistingDefaultsAsync(address.CustomerId,
                    input.IsDefaultBilling,
                    input.IsDefaultShipping,
                    input.IsDefaultDelivery,
                    excludeAddressId: id);
            }

            _mapper.Map(input, address);
            await _addressRepository.UpdateAsync(address);

            var dto = _mapper.Map<Address, AddressDto>(address);

            return new ResponseDataDto<object>
            {
                Success = true,
                Message = "Address updated successfully",
                Data = dto,
                Code = 200
            };
        }
        catch (UserFriendlyException ex)
        {
            _logger.LogWarning(ex, "Validation error: {Message}", ex.Message);
            throw;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Error updating address: {ErrorMessage}", ex.Message);
            throw new UserFriendlyException("An unexpected error occurred. Please try again later.");
        }
    }

    private async Task ClearExistingDefaultsAsync(Guid customerId, bool? billing, bool? shipping, bool? delivery, Guid? excludeAddressId = null)
    {
        var defaultsToClear = new List<Address>();

        var query = _addressRepository.Where(a =>
            a.CustomerId == customerId &&
            a.IsActive &&
            (excludeAddressId == null || a.Id != excludeAddressId.Value));

        if (billing == true)
        {
            var currentDefault = await AsyncExecuter.FirstOrDefaultAsync(
                query.Where(a => a.IsDefaultBilling));
            if (currentDefault != null)
            {
                currentDefault.IsDefaultBilling = false;
                defaultsToClear.Add(currentDefault);
            }
        }

        if (shipping == true)
        {
            var currentDefault = await AsyncExecuter.FirstOrDefaultAsync(
                query.Where(a => a.IsDefaultShipping));
            if (currentDefault != null)
            {
                currentDefault.IsDefaultShipping = false;
                defaultsToClear.Add(currentDefault);
            }
        }

        if (delivery == true)
        {
            var currentDefault = await AsyncExecuter.FirstOrDefaultAsync(
                query.Where(a => a.IsDefaultDelivery));
            if (currentDefault != null)
            {
                currentDefault.IsDefaultDelivery = false;
                defaultsToClear.Add(currentDefault);
            }
        }

        if (defaultsToClear.Any())
        {
            foreach (var addr in defaultsToClear.Distinct())
            {
                await _addressRepository.UpdateAsync(addr);
            }
        }
    }
```

## 19. Additional Considerations

### 19.1 Customer Entity Relationship

The Address entity assumes a one-to-many relationship with the existing Customer entity. Verify the Customer entity has a collection navigation property:

```csharp
// In Products.Entities.Customers.Customer.cs
public virtual ICollection<Address> Addresses { get; set; } = new List<Address>();
```

If not present, add this navigation property to enable efficient queries.

### 19.2 Soft Delete Behavior

ABP's `FullAuditedAggregateRoot<Guid>` includes `IsDeleted` property and automatically filters deleted entities when using repository's `GetQueryableAsync()`. Ensure `IsDeleted` filter is applied in custom queries.

### 39.3 Default Address Uniqueness

The implementation ensures only one default per type (billing, shipping, delivery). The `ClearExistingDefaultsAsync` helper method handles this business rule. Race conditions are mitigated by database transactions (ABP repositories are transactional by default).

### 19.4 Performance

- Index `CustomerId` and `(CustomerId, IsActive)` for fast customer-specific queries
- Consider caching frequently accessed default addresses
- For customers with many addresses (>100), use `GetPagedListAsync` instead of `GetListAsync`
